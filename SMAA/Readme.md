This project Social Media Advertisement Analysis developed in Microsoft Excel, provides a comprehensive analytical view of marketing campaign performance.
It leverages historical campaign data to track user engagement, ad performance across platforms, and key conversion metrics. The dashboard serves as a central hub 
for visualizing how different demographic groups and campaign strategies impact overall marketing efficiency.
Methods Done:
        -Data Modeling: Uses structured data sets—including user records, campaign metadata, and ad event logs—to build a cohesive analytical environment.
        -Pivot Table Integration: Employs advanced pivot tables for dynamic data aggregation and cross-tabulation of multi-dimensional variables (e.g., age vs. gender distribution).
        -KPI Visualization: Synthesizes raw event data into actionable business intelligence metrics, offering a snapshot of campaign ROI and audience outreach.
